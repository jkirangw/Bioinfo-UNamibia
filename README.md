# Bioinfo-Namibia

Teaching students at **The University of Namibia (UNAM)**

## Course overview
This repository contains teaching materials for bioinformatics training, including:
- Lectures
- Practical sessions
- Example datasets
- Scripts and workflows

## Requirements
- Linux (Ubuntu recommended)
- Basic command-line skills
- Git
- Conda / Mamba (for bioinformatics tools)

## Structure
- `lectures/` – lecture slides and notes
- `practicals/` – hands-on exercises
- `data/` – small example datasets
- `scripts/` – analysis scripts
- `assignments/` – student assignments
- `resources/` – references and learning materials
